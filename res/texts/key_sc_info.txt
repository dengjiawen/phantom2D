--- Instructions for creating keyboard shortcuts ---

Phantom2D uses Java platform KeyStroke strings to specify keyboard shortcuts.

String specifying keyboard shortcut must have following syntax:

    <modifiers> <key>

	modifiers := shift | control | ctrl | meta | alt | altGraph 
	key := KeyEvent key code name

Here are some examples:

	"INSERT" => Insert key
	"Y" => y key
	"control DELETE" => Delete key while holding down control key
	"alt shift X" => x key while holding down alt and shift keys
